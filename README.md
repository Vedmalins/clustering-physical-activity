# Кластеризация физической активности человека

## Описание проекта

Проект посвящён решению задачи кластеризации физической активности человека на основе данных носимых сенсоров.

Данные были получены с трёх инерциальных измерительных модулей (IMU), расположенных на руке, груди и лодыжке. Цель проекта — выявить скрытые группы активности с помощью методов обучения без учителя и подготовить submission-файл для соревнования Kaggle.

Проект выполнен в рамках домашнего задания по машинному обучению без учителя.

## Цель проекта

Разработать воспроизводимое решение для кластеризации данных физической активности человека и сравнить несколько алгоритмов кластеризации.

## Что было сделано

В рамках проекта выполнены следующие этапы:

1. Загрузка и первичный анализ данных;
2. Разведочный анализ данных (EDA);
3. Анализ пропусков, выбросов и распределений признаков;
4. Корреляционный анализ признаков и групп сенсоров;
5. Предобработка данных;
6. Feature Engineering;
7. Масштабирование признаков;
8. Снижение размерности методом PCA;
9. Обучение и сравнение нескольких алгоритмов кластеризации;
10. Подбор гиперпараметров;
11. Выбор лучшей модели;
12. Подготовка submission-файла для Kaggle.

## Использованные алгоритмы

В проекте были рассмотрены следующие методы кластеризации:

* K-Means;
* Agglomerative Clustering;
* DBSCAN;
* Gaussian Mixture Model.

## Feature Engineering

Для улучшения качества кластеризации были созданы дополнительные признаки:

* модули ускорений;
* модули гироскопов;
* модули магнитометров;
* средняя температура;
* средняя ориентация сенсоров.

## Лучшая модель

По результатам экспериментов и отправок на Kaggle лучшим решением стала модель:

* K-Means;
* количество кластеров: 12;
* использование `subject_id`;
* PCA: 30 компонент;
* Kaggle Public Score: 0.23320.
