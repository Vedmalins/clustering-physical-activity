# clustering-physical-activity
Кластеризация физической активности человека на основе данных носимых сенсоров (PAMAP2). Исследование методов K-Means, Agglomerative Clustering, DBSCAN и Gaussian Mixture Models.


# Кластеризация физической активности человека

## Описание проекта

Данный проект посвящён задаче кластеризации физической активности человека на основе данных, полученных с носимых сенсоров.

В работе используются данные с трёх инерциальных измерительных модулей (IMU), расположенных на руке, груди и лодыжке, а также дополнительные физиологические показатели. Основная цель проекта — выявление скрытых групп и паттернов активности с помощью методов обучения без учителя.

Проект выполнен в рамках учебного задания по анализу данных и машинному обучению.

---

## Цель исследования

Построить и сравнить различные алгоритмы кластеризации для выявления естественной структуры данных о физической активности человека.

---

## Задачи проекта

* Провести разведочный анализ данных (EDA);
* Исследовать пропуски, выбросы и распределения признаков;
* Выполнить корреляционный анализ сенсорных групп;
* Провести предобработку данных;
* Создать дополнительные информативные признаки (Feature Engineering);
* Выполнить снижение размерности методом PCA;
* Сравнить различные алгоритмы кластеризации;
* Оценить качество кластеризации с использованием метрики Silhouette Score;
* Проанализировать полученные кластеры и их интерпретацию.

---

## Используемые методы

### Предобработка данных

* Заполнение пропусков медианой;
* Масштабирование признаков (StandardScaler, RobustScaler);
* Обработка выбросов;
* Формирование итоговой матрицы признаков.

### Feature Engineering

Были построены дополнительные признаки:

* Модули ускорений (Acceleration Magnitude);
* Модули гироскопов (Gyroscope Magnitude);
* Модули магнитометров (Magnetometer Magnitude);
* Средняя температура датчиков;
* Средняя ориентация датчиков.

### Снижение размерности

* Principal Component Analysis (PCA)

### Алгоритмы кластеризации

* K-Means
* Agglomerative Clustering
* DBSCAN
* Gaussian Mixture Model (GMM)

## Основные результаты

В ходе исследования были протестированы несколько алгоритмов кластеризации.

Наилучшие результаты по метрике Silhouette Score показал алгоритм K-Means, который был выбран в качестве основной модели кластеризации.

Проведённый анализ позволил выявить выраженную структуру в пространстве признаков и выделить несколько устойчивых групп наблюдений, соответствующих различным режимам физической активности.
